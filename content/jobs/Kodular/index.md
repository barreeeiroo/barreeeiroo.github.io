Kodular is an online platform to create Android apps without coding. It started as a part-time project with other
highschool students around the globe. It is now a US company that has partnered with companies like Google. Some
of the most important work done is listed below:

- Created a full Service-as-a-Service platform, by developing several web applications using Java (GWT) and Python
  (Django).
- Managing cloud infrastructure in GCP and AWS.
- Acquired AppyBuilder and merged into a single platform, and partnered with companies like Google and Start.io.

_Chief Technology Officer until 2019._
